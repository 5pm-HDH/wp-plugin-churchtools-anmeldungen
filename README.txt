Ermöglicht Darstellung der ChurchTools-Anmeldungen (GroupHomepage) auf der WordPress-Seite mit eigenem Design.

== Description ==

Die Darstellung der ChurchTools-Anmeldungen (GroupHomepage) wird mittels Templates umgesetzt.

== Installation ==

Nach dem Aktivieren der Plugins, musst du in der Seitenleiste die Einstellungen für das Plugin öffnen und deine ChurchTools URL hinterlegen. Danach kannst du die Anmeldungen mittels ShortCode auf deiner Seite einbinden.
